Start Here
  • No membership, completely free: Business Builders, LeanIX, XU Sustainability, Snap!, MicroBlocks
  • Largest capacity: S/4HANA Processes (1,000 students)
  • Shortest modules (8 h): SuccessFactors, Signavio, Build Code
  • Train-the-Lecturer available for all SAP UA full member assets
Combine These Assets
  • Full ERP curriculum: S/4HANA Processes + Configuration (same system)
  • Data pipeline: Datasphere → SAC Magdeburg
  • Build progression: Build Apps → Build Code → BTP Pro-Code
  • Add sustainability layer to S/4HANA, SAC or ERPsim at no extra cost
No SAP Background? Start With
  • Business Builders — zero setup, gamified, 6 scenarios
  • XU Sustainability — self-paced certificate, worldwide
  • SAP LeanIX — register directly, no membership
  • ERPsim — simulation game, intuitive for students
  • Snap! / MicroBlocks — schools & college intro courses
Before You Decide
  • All assets include a Train-the-Lecturer programme (except XU self-paced)
  • SAP BTP Pro-Code available in EMEA & APJ only
  • Two SAC variants: Magdeburg (planning + GBI) vs Milwaukee (statistics + R)
